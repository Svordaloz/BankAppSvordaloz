# BankAppSvordaloz
BankApp med enkel display, i java. Helt CLI-baserad. Fyra funktioner, med tre möjliga valutor att välja mellan för användare. 

## Hur man använder appen
* Se till att du har [Java](https://www.java.com/en/download/) installerad på din dator eller annan lämplig utrustning.
* Kör BankApp.java i egen vald miljö. 
* Kör appen och följ de givna instruktionerna.

## Allmän beskrivning
Applikationen låter användaren sätta in och ta ut pengar, samt visa sitt nuvarande saldo. Kan också hantera tre olika länders valutor; usd, tysk euro och svenska kronor. 
Kan avslutatas utan att ta något annat beslut. 

## Funktioner i appen
* Interaktiva menyer i konsol. 
* Kan välja mellan 3 olika valutor, usd, svensk krona och tysk euro. 
* Balance checking - kan kolla saldo. 
* Kan hantera utdrag och insättningar i olika valutor. 
* Varningsmeddelande för användaren om de gör en selektion som inte är passlig inom appens funktionalitet.
